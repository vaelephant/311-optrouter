
🧠 OptRouter 是什么

OptRouter 是一个 统一的大模型 API 网关，可以让你通过一个接口调用多家 AI 模型。

👉 官方定位：

One API → Access multiple LLM providers

你可以把它理解为：

✅ AI模型的「统一入口」
✅ 类似 Stripe（支付聚合）但用于 AI
✅ 类似 Cloudflare Gateway，但针对 LLM

添加新 Provider 现在只需 3 步：
新建 providers/mistral.rs，实现 Provider trait
在 providers/mod.rs 的 build_provider() 加一个 match 分支
在 router/mod.rs 的路由表里登记对应模型
handlers/chat.rs 从原来的 767 行精简到了 235 行，全部是流程逻辑，不含任何 Provider 细节。
