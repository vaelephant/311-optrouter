# Gateway 快速启动

## 前置条件

- 已配置 `.env`（含 `DATABASE_URL`、`REDIS_URL` 等）
- Postgres、Redis 已启动
- 模型配置存在：`config/models.toml`（或在 `.env` 里设置 `REGISTRY_CONFIG`）

## 编译

```bash
cd gateway
cargo build
```

## 启动

```bash
cd gateway
cargo run
```

默认端口：`3001`（可用 `PORT` 覆盖）。

## 重启（先杀再起）

```bash


lsof -ti:3001 | xargs kill -9 2>/dev/null; pkill -f gateway 2>/dev/null; clear; cargo run


```
