# 计费测试

## 前提

`model_pricing` 表里已为模型配置单价且 `enabled = true`。例如：

| model                   | input_price | output_price | provider |
|-------------------------|-------------|--------------|----------|
| gpt-4o                  | 0.005       | 0.015        | openai   |
| qwen2.5:7b-instruct     | 0.1         | 0.2          | ollama   |

## 1. 测 Ollama 扣费

```bash
curl -X POST http://localhost:3001/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"model":"qwen2.5:7b-instruct","messages":[{"role":"user","content":"说10个字"}],"stream":false}'
```

## 2. 测 OpenAI 扣费

```bash
curl -X POST http://localhost:3001/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"说4个字"}],"stream":false}'
```

## 3. 观察是否扣费

- **用户余额**：查 `user_balances`，看对应用户的 `balance` 是否减少。
- **流水**：查 `usage_logs`、`transactions`，应有新记录且 `cost`/`amount` 与定价一致。
